## рҹҡҖ What Is TaskFlow Pro?

Most task management tools are built for *one* kind of team. **TaskFlow Pro is built for every team.**

Inspired by the flexibility of Google Classroom, the execution model of LeetCode, and the dashboard power of Jira вҖ” TaskFlow Pro fills a gap no existing platform does: **universal, domain-agnostic task management with an integrated code compiler and AI-powered deliverable feedback.**

A manager can:
- рҹҗӣ Assign a software engineer to **fix a bug** вҖ” with a built-in compiler to write & run code
- рҹҺЁ Ask a designer to **create an infographic** вҖ” with file upload and AI design feedback  
- рҹ“„ Instruct HR to **draft a policy document** вҖ” with writing quality analysis
- рҹ“Ҡ Request a finance intern to **complete a spreadsheet analysis** вҖ” all in one place

**All from the same interface. All with AI feedback. All tracked in one dashboard.**

## рҹ“җ Key Requirements at a Glance

### Functional
- вң… User registration with bcrypt password hashing
- вң… JWT tokens (24hr validity, refreshable on active session)
- вң… Unique 8-character alphanumeric team invite codes
- вң… Task schema: `task_id`, `title`, `description`, `task_type_tag`, `deadline`, `assigned_to`, `status`, `attached_materials`
- вң… Task statuses: `Pending вҶ’ In Progress вҶ’ Submitted вҶ’ Reviewed` (auto `Overdue` when deadline passes)
- вң… File uploads up to 50MB per submission/material
- вң… Sandboxed code execution (stdout, stderr, execution time)
- вң… AI feedback: quality score (0вҖ“100), key issues, improvement suggestions
- вң… Manager dashboard with per-member completion rates

### Non-Functional
- вҡЎ **Performance**: API responds to 95% of requests within 500ms at 200 req/s
- рҹ”’ **Security**: bcrypt hashing + HTTPS/TLS + sandboxed code execution (no network access)
- рҹ“Ҳ **Scalability**: Modular, horizontally scalable вҖ” compiler, ML, and core services scale independently
- рҹӣЎпёҸ **Reliability**: Automated DB backups every 24 hours, point-in-time recovery within 1 hour
- рҹҢҗ **Compatibility**: Chrome, Firefox, Safari, Edge (latest 2 major versions) on desktop & mobile
- рҹ§Ә **Maintainability**: вүҘ70% unit test coverage on all backend service modules

## рҹ“Ӣ Use Cases

<details>
<summary><strong>UC-01: User Registration</strong></summary>

**Flow**: User provides username, email, password вҶ’ system validates вҶ’ bcrypt hashes password вҶ’ account created вҶ’ redirected to login.

**Validations**: Valid email format, 8+ char password with uppercase + number, unique email check.

<details>
<summary><strong>UC-02: Create or Join a Team</strong></summary>

**Create**: Enter team name + domain вҶ’ 8-char invite code generated вҶ’ user becomes Manager.  
**Join**: Enter invite code вҶ’ validated вҶ’ user added as Team Member вҶ’ redirected to workspace.

</details>

<details>
<summary><strong>UC-03: Assign a Task</strong></summary>

**Manager flow**: Fill title, description, task type tag, deadline, select members, optionally attach materials вҶ’ system validates вҶ’ task saved вҶ’ in-app notifications sent to assigned members.

</details>

<details>
<summary><strong>UC-04: Submit Task Deliverable</strong></summary>

**Member flow**: View task вҶ’ prepare deliverable (code / file / text / link) вҶ’ optionally run code вҶ’ submit вҶ’ AI generates feedback (score + issues + suggestions) вҶ’ status updated to Submitted.

</details>

<details>
